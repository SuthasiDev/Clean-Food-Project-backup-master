# Clean-Food-Project-backup

โปรเจ็คแอนดรอย Clean Food

สำหรับยูสเซอร์ทั่วไป
- แสดงเมนูสำหรับอาหารคลีน เป็น grid view (รูปภาพ ชื่อเมนู) คลิกเข้าไปจะแสดงรายละเอียดของเมนู (รูปภาพ ชื่อเมนู ส่วนผสม วิธีการทำ รายละเอียด) สามารถกดเป็นเมนูโปรดได้ สามารถคอมเม้นท์ได้ ดูคอมเม้นท์ได้
- เมนูโปรด แสดงเป็น grid view (รูปภาพ ชื่อเมนู)
- แสดงร้านอาหารสำหรับอาหารคลีน เป็น list view (รูปภาพ ชื่อร้าน) คลิกเข้าไปจะแสดงรายละเอียดของร้าน (รูปภาพ ชื่อร้าน รายละเอียด พิกัด เบอร์โทร เว็บไซต์) สามารถกดเป็นร้านโปรดได้ สามารถคอมเม้นท์ได้ ดูคอมเม้นท์ได้
- ร้านโปรด แสดงเป็น list view (รูปภาพ ชื่อร้าน)

สำหรับ Log In ด้วย Admin
- Form สำหรับเพิ่มเมนูอาหารคลีน
- Form สำหรับเพิ่มร้านอาหารคลีน
